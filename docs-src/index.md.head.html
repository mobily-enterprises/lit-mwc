Powerful, themeable, enhanced native elements

The Platform Elements (TPE) is not a custom implementation of native elements. It is native elements, enhanced. TPE form controls are 100% customizable, and still behave like you would expect from native elements, while also providing additional capabilities.

From this...

<label for="native"><span>Native Input</span></label>
<input id="native"  pattern="[a-z]*" />

...to this!

<nn-input-text label="TPE Input" pattern="[a-z]*"></nn-input-text>
